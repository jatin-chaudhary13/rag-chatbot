# 🤖 Wattmonk Chatbot 

## 🚀 Overview
This project is an AI-powered chatbot built using:
- Retrieval-Augmented Generation (RAG)
- Groq LLM (LLaMA 3)
- Streamlit UI

It answers:
- General questions
- NEC-related queries
- Wattmonk company information

---

## 🛠️ Tech Stack
- Python
- Streamlit
- LangChain
- FAISS
- HuggingFace Embeddings
- Groq API
