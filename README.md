# 🤖 Wattmonk Chatbot — AI RAG Assistant

## 🚀 Overview
Wattmonk Chatbot is an AI-powered assistant built using Retrieval-Augmented Generation (RAG).  
It can answer both general questions and domain-specific queries using custom PDF documents (NEC & Wattmonk).

The chatbot combines LLMs, semantic search, and vector databases to provide accurate and context-aware responses.

---

## 🌐 Live Demo
https://wattmonkchatbot.streamlit.app/

---

## ✨ Features
- ChatGPT-like conversational interface  
- Answers general + document-based queries  
- Uses PDF knowledge (NEC & Wattmonk)  
- Displays source references  
- Confidence scoring for responses  
- Suggested follow-up questions  
- Multi-chat support  

---

## 🛠 Tech Stack
- Frontend: Streamlit  
- LLM: Groq (LLaMA 3)  
- Embeddings: HuggingFace (MiniLM)  
- Vector Database: FAISS  
- Framework: LangChain  
- Language: Python
